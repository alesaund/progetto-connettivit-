# ADR-0001 - Filosofia del progetto

## Stato
Accettato

## Contesto
L'obiettivo è costruire una rete domestica affidabile, documentata e facilmente manutenibile, adottando pratiche tipiche di ambienti enterprise.

## Decisioni
- Documentazione come codice (Docs as Code).
- GitHub come fonte ufficiale della documentazione.
- Runbook per ogni attività operativa.
- Ogni modifica significativa deve essere tracciata.
- Le decisioni progettuali vengono documentate tramite ADR.

## Conseguenze
La manutenzione futura sarà ripetibile, verificabile e indipendente dalla memoria dell'amministratore.
