# Progetto Connettività

Questo repository contiene la documentazione tecnica dell'infrastruttura di rete.

## Struttura

```text
docs/
│
├── architecture/      # Topologia, piano IP, Wi-Fi, hardware
├── inventory/         # Schede dei dispositivi
├── runbooks/          # Procedure operative (SOP)
├── troubleshooting/   # Analisi e casi risolti
└── changelog/         # Storico delle modifiche
```

## Obiettivi

- Documentazione versionata
- Runbook ripetibili
- Change management
- Troubleshooting storico
- Disaster recovery

Nei prossimi aggiornamenti ogni sezione verrà popolata progressivamente con la documentazione dell'infrastruttura OpenWrt e dei progetti correlati (Project Nomad compreso dove pertinente).
